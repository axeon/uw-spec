# 移动端设计规范

> 移动端设计规范，被 220-md-uniapp-design 引用。

## 色彩规范

```css
--primary-color: #409EFF;      /* 主题色 */
--success-color: #67C23A;      /* 成功色 */
--warning-color: #E6A23C;      /* 警告色 */
--danger-color: #F56C6C;       /* 危险色 */
--text-primary: #303133;       /* 主要文字 */
--text-regular: #606266;       /* 常规文字 */
--border-color: #DCDFE6;       /* 边框颜色 */
--bg-color: #F5F7FA;           /* 背景颜色 */
```

## 字体规范

| 级别 | 大小 | 用途 |
|------|------|------|
| 标题 | 18px | 页面标题 |
| 小标题 | 16px | 模块标题 |
| 正文 | 14px | 常规内容 |
| 辅助 | 12px | 辅助信息 |

## 间距规范

| 类型 | 间距 |
|------|------|
| 基础间距 | 8px |
| 模块间距 | 16px/24px |
| 页面边距 | 16px |

## 平台适配

| 平台 | 设计要求 |
|------|---------|
| H5 | 响应式设计，适配各种屏幕 |
| 小程序 | 遵循微信小程序设计规范 |
| App | iOS/Android原生体验 |

## 设计尺寸

- 设计基准: 375px (iPhone 6/7/8)
- 安全区域: 考虑刘海屏、圆角屏

## 移动端专项

| 要求 | 规范 |
|------|------|
| 点击区域 | ≥ 44x44px |
| 手势支持 | 下拉刷新、上拉加载、左滑删除、右滑返回 |
| 屏幕适配 | 刘海屏/圆角屏、横竖屏切换、系统字体大小 |
| 性能优化 | 图片懒加载、列表虚拟滚动、页面缓存策略 |
